## 🎯 功能使用

### 1. 录音转文字

#### 步骤

1. **打开应用**
   - 首次使用会请求麦克风和存储权限
   - 点击"允许"授予权限

2. **开始录音**
   - 点击主界面右下角的录音按钮（麦克风图标）
   - 进入录音界面

3. **选择方言**（可选）
   - 点击"选择方言"按钮
   - 从列表中选择合适的方言
   - 支持：普通话、粤语、四川话等 12 种方言

4. **开始录制**
   - 点击"开始录音"按钮
   - 开始说话，应用会实时显示识别结果
   - 临时结果显示在上方（斜体）
   - 确定结果显示在下方

5. **暂停/继续**
   - 需要暂停时点击"暂停"按钮
   - 继续录音点击"继续"按钮

6. **停止录音**
   - 点击"停止录音"按钮
   - 录音和识别结束

7. **保存记录**
   - 点击"保存"按钮
   - 输入会议标题
   - 点击"保存"完成

#### 注意事项

- ✅ 在安静的环境中录音效果更好
- ✅ 说话清晰，语速适中
- ✅ 选择正确的方言类型
- ✅ 确保网络连接正常（实时识别需要网络）
- ⚠️ 长时间录音会消耗较多流量

### 2. 导入音频文件

#### 步骤

1. **选择导入**
   - 在主界面点击导入按钮（向上箭头图标）
   - 或点击菜单中的"导入音频"

2. **选择文件**
   - 从文件管理器中选择音频文件
   - 支持格式：MP3、WAV、M4A、AAC 等

3. **等待识别**
   - 应用会自动上传并识别音频
   - 显示识别进度

4. **查看结果**
   - 识别完成后自动保存
   - 可以在列表中查看

#### 支持的音频格式

- WAV（推荐）
- MP3
- M4A
- AAC
- FLAC

#### 注意事项

- ✅ 音频文件不宜过大（建议 < 100MB）
- ✅ 音质越好，识别准确率越高
- ✅ 需要网络连接
- ⚠️ 识别时间取决于音频长度

### 3. 编辑会议内容

#### 步骤

1. **打开记录**
   - 在主界面点击任意会议记录
   - 进入编辑界面

2. **编辑标题**
   - 点击标题输入框
   - 修改会议标题

3. **编辑内容**
   - 在内容区域编辑文字
   - 支持多行文本
   - 可以复制、粘贴

4. **保存更改**
   - 点击"保存"按钮
   - 更改会自动保存到数据库

#### 快捷操作

- **长按文本**：选择、复制、粘贴
- **双击**：选择单词
- **三击**：选择段落

### 4. AI 智能总结

#### 步骤

1. **打开编辑界面**
   - 点击要总结的会议记录

2. **生成摘要**
   - 点击"生成摘要"按钮
   - 选择 AI 模型：
     - **OpenAI GPT**：准确度高，需要 API Key
     - **通义千问**：国内访问快，需要 API Key
     - **文心一言**：百度出品，需要 API Key
     - **本地总结**：无需 API，功能简单

3. **等待生成**
   - 显示"生成中..."
   - 通常需要 5-30 秒

4. **查看摘要**
   - 摘要会显示在摘要区域
   - 包含：
     - 会议主题
     - 主要议题
     - 关键决策
     - 行动项
     - 其他要点

5. **编辑摘要**
   - 可以手动修改摘要内容
   - 点击"保存"保存更改

#### 配置 AI API

在设置中配置 API Key：

1. 进入"设置" → "AI 总结"
2. 选择要使用的 AI 模型
3. 输入对应的 API Key
4. 点击"测试连接"验证
5. 保存设置

#### 获取 API Key

- **OpenAI**: https://platform.openai.com/api-keys
- **通义千问**: https://dashscope.console.aliyun.com/apiKey
- **文心一言**: https://console.bce.baidu.com/qianfan/ais/console/applicationConsole/application

#### 注意事项

- ✅ 内容越详细，摘要越准确
- ✅ 建议使用付费 API 获得更好效果
- ✅ 注意 API 调用次数限制
- ⚠️ 敏感内容请谨慎使用在线 API

### 5. 导出文档

#### 步骤

1. **打开编辑界面**
   - 点击要导出的会议记录

2. **选择导出**
   - 点击"导出文档"按钮
   - 选择导出格式：
     - **TXT**：纯文本，兼容性最好
     - **Markdown**：支持格式化，适合技术文档
     - **Word**：.docx 格式，适合办公
     - **PDF**：便于分享和打印
     - **HTML**：网页格式，可在浏览器查看

3. **等待导出**
   - 显示导出进度
   - 通常只需几秒钟

4. **分享文件**
   - 导出完成后自动弹出分享菜单
   - 可以分享到：
     - 微信
     - QQ
     - 钉钉
     - 邮件
     - 云盘
     - 其他应用

#### 导出位置

文件保存在：
```
/Android/data/com.meetingminutes.app/files/exports/
```

#### 各格式特点

| 格式 | 优点 | 缺点 | 适用场景 |
|------|------|------|----------|
| TXT | 体积小，兼容性好 | 无格式 | 纯文本记录 |
| Markdown | 支持格式，易编辑 | 需要 MD 阅读器 | 技术文档 |
| Word | 格式丰富，通用 | 体积较大 | 正式文档 |
| PDF | 格式固定，通用 | 不易编辑 | 分享、打印 |
| HTML | 可在浏览器查看 | 需要浏览器 | 网页展示 |

#### 注意事项

- ✅ 导出前确保内容已保存
- ✅ PDF 导出需要中文字体支持
- ✅ Word 文档可在 WPS、Office 中打开
- ⚠️ 大文件导出可能需要较长时间

### 6. 搜索记录

#### 步骤

1. **打开搜索**
   - 在主界面点击搜索图标
   - 或下拉列表触发搜索框

2. **输入关键词**
   - 输入要搜索的关键词
   - 支持搜索标题和内容

3. **查看结果**
   - 实时显示匹配的记录
   - 关键词会高亮显示

4. **打开记录**
   - 点击搜索结果打开记录

#### 搜索技巧

- 使用空格分隔多个关键词
- 搜索会匹配标题和内容
- 不区分大小写

### 7. 删除记录

#### 步骤

1. **选择记录**
   - 在列表中找到要删除的记录

2. **点击删除**
   - 点击记录右侧的删除按钮（垃圾桶图标）

3. **确认删除**
   - 弹出确认对话框
   - 点击"删除"确认
   - 点击"取消"放弃

#### 注意事项

- ⚠️ 删除操作不可恢复
- ⚠️ 删除记录会同时删除关联的音频文件
- ✅ 建议重要记录先导出备份

### 8. 版本更新

#### 自动检查

应用会在启动时自动检查更新：

1. 发现新版本时弹出提示
2. 显示更新日志
3. 选择操作：
   - **立即更新**：下载并安装
   - **稍后提醒**：下次启动再提醒
   - **忽略此版本**：不再提醒此版本

#### 手动检查

1. 进入"设置" → "关于"
2. 点击"检查更新"
3. 查看当前版本和最新版本

#### 更新流程

1. 点击"立即更新"
2. 下载 APK 文件（显示进度）
3. 下载完成后自动打开安装界面
4. 点击"安装"完成更新

## 🔧 故障排除

### 录音没有声音

**可能原因**：
- 未授予麦克风权限
- 麦克风被其他应用占用
- 硬件问题

**解决方案**：
1. 检查权限设置
2. 关闭其他使用麦克风的应用
3. 重启应用
4. 重启手机

### 识别不准确

**可能原因**：
- 环境噪音大
- 方言选择错误
- 网络不稳定
- 说话不清晰

**解决方案**：
1. 在安静环境录音
2. 选择正确的方言
3. 检查网络连接
4. 说话清晰、语速适中

### 导出失败

**可能原因**：
- 存储空间不足
- 权限问题
- 内容过大

**解决方案**：
1. 清理存储空间
2. 检查存储权限
3. 分段导出大文件

### 应用崩溃

**解决方案**：
1. 清除应用缓存
2. 重新安装应用
3. 检查系统版本（需要 Android 7.0+）
4. 提交 Bug 报告

## 💡 使用技巧

### 提高识别准确率

1. **环境**：选择安静的环境
2. **距离**：手机距离嘴巴 20-30cm
3. **语速**：适中，不要太快
4. **发音**：清晰，避免口音过重
5. **方言**：选择正确的方言类型

### 高效整理会议

1. **实时录音**：会议进行时实时录音
2. **快速总结**：会后立即生成 AI 摘要
3. **及时编辑**：趁记忆清晰时补充细节
4. **分类管理**：使用标签分类
5. **定期导出**：重要会议导出备份

### 节省流量

1. **使用 WiFi**：尽量在 WiFi 环境下使用
2. **离线识别**：考虑使用离线识别引擎
3. **批量处理**：积累多个文件后统一处理

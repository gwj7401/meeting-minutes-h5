# 📤 导出功能使用说明

## 功能概述

会议纪要应用现在支持将会议记录导出为多种格式的文档，方便分享和存档。

---

## 支持的导出格式

### 1. TXT 文本格式 📄

**特点：**
- ✅ 纯文本格式
- ✅ 兼容性最好
- ✅ 文件体积最小
- ✅ 可在任何文本编辑器打开

**适用场景：**
- 简单的文字记录
- 需要复制粘贴到其他应用
- 存档备份

**示例内容：**
```
会议记录 2025/11/2 13:55:19
创建时间: 2025/11/2 13:55:19
时长: 0:17
语言: 普通话

会议内容:
好现在开始开会啦，首先能够出去玩儿，看我说的对不对？...
```

---

### 2. Markdown 格式 📝

**特点：**
- ✅ 支持格式化
- ✅ 可在 GitHub、Notion 等平台查看
- ✅ 易于编辑和版本控制
- ✅ 支持转换为其他格式

**适用场景：**
- 需要格式化的文档
- 上传到 GitHub/GitLab
- 导入到 Notion、Obsidian 等笔记应用
- 技术文档

**示例内容：**
```markdown
# 会议记录 2025/11/2 13:55:19

**创建时间**: 2025/11/2 13:55:19  
**时长**: 0:17  
**语言**: 普通话

## 会议内容

好现在开始开会啦，首先能够出去玩儿，看我说的对不对？...
```

---

### 3. HTML 格式 🌐

**特点：**
- ✅ 精美的排版
- ✅ 可在浏览器直接打开
- ✅ 支持打印
- ✅ 可转换为 PDF

**适用场景：**
- 需要打印的文档
- 发送给客户或领导
- 存档展示
- 转换为 PDF

**特色：**
- 紫色主题设计
- 响应式布局
- 优雅的排版
- 适合打印

---

## 如何导出

### 方法 1：从会议详情页导出

1. **打开会议详情**
   - 在首页点击任意会议记录
   - 进入会议详情页

2. **点击导出按钮**
   - 点击右上角的"分享"图标 📤
   - 会弹出导出菜单

3. **选择导出格式**
   - TXT 文本
   - Markdown
   - HTML

4. **保存文件**
   - 浏览器会自动下载文件
   - 文件名为会议标题

---

### 方法 2：从首页快速导出

1. **在首页找到会议记录**
   - 浏览会议列表

2. **左滑会议项**
   - 向左滑动会议项
   - 会显示"导出"和"删除"按钮

3. **点击导出按钮**
   - 点击蓝色的"导出"按钮
   - 弹出导出菜单

4. **选择格式并导出**
   - 选择需要的格式
   - 自动下载

## 导出后的操作

### TXT 文件

**打开方式：**
- Windows: 记事本、Notepad++、VS Code
- Mac: 文本编辑、VS Code
- 手机: 任何文本编辑器

**后续操作：**
- 复制粘贴到邮件
- 上传到云盘
- 导入到其他应用

---

### Markdown 文件

**打开方式：**
- Typora（推荐）
- VS Code + Markdown 插件
- Notion（导入）
- Obsidian
- GitHub/GitLab（直接查看）

**后续操作：**
- 上传到 GitHub
- 导入到 Notion
- 转换为 PDF
- 继续编辑

**转换为 PDF：**
1. 使用 Typora 打开
2. 文件 → 导出 → PDF
3. 保存

---

### HTML 文件

**打开方式：**
- 任何浏览器（Chrome、Edge、Safari 等）
- 双击文件即可打开

**后续操作：**

#### 1. 打印为 PDF

**Chrome/Edge:**
1. 打开 HTML 文件
2. 按 `Ctrl + P`（Mac: `Cmd + P`）
3. 目标打印机选择"另存为 PDF"
4. 点击"保存"

**效果：**
- 保留所有格式和样式
- 适合存档和分享

#### 2. 直接打印

1. 打开 HTML 文件
2. 按 `Ctrl + P`（Mac: `Cmd + P`）
3. 选择打印机
4. 点击"打印"

#### 3. 发送给他人

- 直接发送 HTML 文件
- 对方用浏览器打开即可查看
- 无需安装任何软件

## 文件命名规则

导出的文件会自动使用会议标题作为文件名：

```
会议记录 2025/11/2 13:55:19.txt
会议记录 2025/11/2 13:55:19.md
会议记录 2025/11/2 13:55:19.html
```

**注意：** 如果文件名包含特殊字符，浏览器可能会自动替换。

## 批量导出

目前不支持批量导出，但可以：

1. **逐个导出**
   - 在首页左滑每个会议
   - 点击"导出"按钮

2. **使用浏览器开发者工具**
   - 按 F12 打开控制台
   - 运行批量导出脚本（高级用户）

## 常见问题

### Q1: 导出的文件在哪里？

**A:** 在浏览器的默认下载文件夹中。

- Chrome: 设置 → 下载内容 → 位置
- Edge: 设置 → 下载 → 位置

### Q2: 可以修改导出的文件名吗？

**A:** 可以。下载后在文件管理器中重命名即可。

### Q3: HTML 文件可以在手机上打开吗？

**A:** 可以。使用任何浏览器打开即可。

### Q4: 如何将 HTML 转换为 PDF？

**A:** 
1. 用浏览器打开 HTML 文件
2. 按 `Ctrl + P` 打印
3. 选择"另存为 PDF"
4. 保存

### Q5: Markdown 文件如何查看？

**A:** 
- 在线：上传到 GitHub 或 GitLab
- 本地：使用 Typora、VS Code 等编辑器
- 导入：导入到 Notion、Obsidian 等笔记应用

### Q6: 导出的文件包含录音吗？

**A:** 不包含。导出的文件只包含文字内容和元数据（标题、时间、时长等）。

录音文件保存在浏览器的 IndexedDB 中，如需导出录音，可以在详情页播放录音时使用录屏软件录制。

### Q7: 可以导出为 Word 格式吗？

**A:** 目前不直接支持 Word 格式，但可以：

**方法 1：** 通过 Markdown
1. 导出为 Markdown
2. 使用 Pandoc 转换为 Word
   ```bash
   pandoc input.md -o output.docx
   ```

**方法 2：** 通过 HTML
1. 导出为 HTML
2. 用 Word 打开 HTML 文件
3. 另存为 .docx 格式

**方法 3：** 复制粘贴
1. 在详情页点击"复制"按钮
2. 粘贴到 Word 中
3. 手动调整格式

## 高级技巧

### 1. 自定义 HTML 样式

导出的 HTML 文件可以手动编辑样式：

1. 用文本编辑器打开 HTML 文件
2. 找到 `<style>` 标签
3. 修改 CSS 样式
4. 保存并刷新浏览器

### 2. 批量转换为 PDF

使用命令行工具批量转换：

```bash
# 安装 wkhtmltopdf
# Windows: choco install wkhtmltopdf
# Mac: brew install wkhtmltopdf

# 批量转换
for file in *.html; do
  wkhtmltopdf "$file" "${file%.html}.pdf"
done
```

### 3. 导入到 Notion

1. 导出为 Markdown
2. 在 Notion 中创建新页面
3. 点击"导入"
4. 选择 Markdown 文件
5. 上传
